# MyFirstAlexaSkill
Python Code for building a weather bot using Alexa
Before using the code, pls add in your Key from weatherstack.com 
In the method "get_InquireWeather_response" you will need to provide this key in the Parms column also please check your Intent names. 
Custom Intent names that were used for this codebase are "greet", "goodbye" and "InquireWeather"
